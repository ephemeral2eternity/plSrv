All Video Server's Information

All video servers' agents are running on default port: 8615 (control plane). Following apps only discover cache agents running on port 8615! The default video streaming port is 80 (data plane).

  • Obtain all video servers deployed in Google Cloud that run cache agents on port 8615.
  • Notify all video servers about the new manager.
  • Initialize the cache agent overlay network for all video servers.
  • Query the cache agent overlay network from all video servers.
  • Overlay Query: query the existing overlay network of video servers
  • Overlay Query: show the node info in the cache agent overlay network
  • Overlay Query: show the edges in the cache agent overlay network
  • Overlay Query: show all the node's datacenter zones
  • Overlay Graph: show the overlay graph plotted in D3.js

The closest cache agent for recent clients

Whenever a client request comes to the VoD system, it will be served by the closest cache agent (redirector) in the system. Following operations are REST API for showing, adding, deleting, or dumping such information

  • Showing all recent clients and their cache agents.
  • Add one record by (Usually called by clients): /cacheagent/add?client=clientName&cache_agent=video_srv_name
  • Dump all recent clients' cache agents into json file in /tmp/cache_agents_timestampe.json
  • Delete all recent clients' cache agents info.
